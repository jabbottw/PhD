Description:

"file2pdf" is a command line program that converts TIFF, JPEG and text files to PDF (Portable Document Format) files (for a Windows interface to this program, see wdavince). This program is available only to DaVince Tools Plus users.  The key features of this program are:
  • Combines all the features of tiff2pdf, jpeg2pdf and txt2pdf
  • Create PDF files from multiple file types
  • Utilizes the DaVince Tools DLL library, a PDF generating module that can be utilized by programmers to add PDF generation to their programs.
  • PDF file compression support
  • Automatic bookmark creation option
  • Convert many files to one PDF, a directory of files to one PDF, or create a PDF file for each file
  • Individual filenames with wildcard support and/or an entire directory may be specified for conversion
  • Specify files or directories to convert either via the command line or a command file
  • Large file support (1,000+ pages)
  • Creates a version 1.2 PDF file with a unique file ID in trailer
The initialization file and the use of profiles is described in the "DaVince Tools" Converters page. Since this converter combines the features of tiff2pdf, jpeg2pdf and txt2pdf, profile settings from these converters can be combined into a single profile.Common profile settings as well as converter specific settings can be used in a profile.

Common Profile Settings

Refer to the "DaVince Tools" Converters page for a description of the common profile settings.

Converter Specific Settings

The special profile name "_converter" is used to define the converter to use for each recognized file type. By default, the profile is defined as follows:

[_converter]
tiff=tiff,tif
jpeg=jpeg,jpg
text=txt,ini,bat

The default assigns the file types to the three converters. For example, all files ending in ".txt" will be converted using the text converter.  If a file type is encountered that is not defined by any converter, it is skipped. File types can be added to one of the converters if one wishes to include them in conversion.

A default profile is defined, however, most users will want to create their own profiles that combine profile settings of other converters.

Additional Information:

Any tool that creates a PDF must create an intermediate file prior to writing the PDF file. The filename of this intermediate file begins with "tmps" and appears in the directory pointed to by the environment variable "TMP". If this environment does not exist, then the temporary file is written in the current directory. The intermediate file exists only during the execution of the program.
